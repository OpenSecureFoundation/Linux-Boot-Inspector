# Conception et mise en œuvre de Linux Boot Inspector, un outil d’analyse du démarrage d’un distribution Linux: cas de Manjaro

# Objectifs:

1. Comprendre les étapes du boot 

2. Analyser les évènements du boot

3. Détecter les problèmes de démarrage (services lents, erreurs).

4. Étudier le temps de boot global et par service
